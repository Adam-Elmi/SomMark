# Getting Started with SomMark

Welcome! This guide will help you create and convert your first SomMark document in just a few minutes.

## 1. Quick Installation

You don't need to install anything to try it out. Use `npx` to run SomMark instantly:

```bash
npx sommark --version
```

## 2. Your First Document

Create a file named `hello.smark` and add this content:

```ini
[h1]Hello SomMark[end:h1]

Welcome to my first structured document!

[a = href: "https://sommark.org"]Check out our site[end:a]
```

## 3. Convert Your File

Turn your `.smark` file into HTML:

```bash
sommark --html hello.smark
```

*This creates `hello.html` in your folder.*

---

## 4. Start a Project

For larger projects, create a config file so you don't have to retype your settings every time:

```bash
sommark init
```

This creates a `smark.config.js` file where you can save your settings.

---

## 5. Using SomMark in Code

```javascript
import SomMark from "sommark";

const sm = new SomMark({ 
    src: "[h1]Hello SomMark[end:h1]", 
    format: "html" 
});

const result = await sm.transpile();
console.log(result);
// <h1>Hello SomMark</h1>
```
