# Getting Started with SomMark

Welcome! This guide will help you create and convert your very first SomMark document in just a few minutes.

## 1. Quick Installation

You don't need to install anything to try it out. Use `npx` to run SomMark instantly:

```bash
npx sommark --version
```

## 2. Your First Document

Create a new file named `hello.smark`. Open it and add this simple content:

```ini
[h1]Hello SomMark[end]

Welcome to my first structured document!
(Check out our site)->(link: "https://sommark.org")
```

## 3. Convert Your File

Now, use the CLI to turn your `.smark` file into a web page (`.html`):

```bash
sommark --html hello.smark
```

*This creates a new file called `hello.html` in your folder.*

---

## 4. Pro Tip: Start a Project

If you are working on a larger project, the best way to start is by creating a configuration file:

```bash
sommark init
```

This creates a `smark.config.js` file where you can save your settings so you don't have to type them every time.

---

## 5. Using SomMark in Your Code

If you are a developer building an app, you can use the SomMark engine directly in your JavaScript code:

```javascript
import SomMark from "sommark";

// 1. Your source text
const source = "[h1]Hello SomMark[end]";

// 2. Initialize the engine
const sm = new SomMark({ 
    src: source, 
    format: "html" 
});

// 3. Convert it!
const result = await sm.transpile();

console.log(result); // Outputs: <h1>Hello SomMark</h1>
```
