# Installation

You can install SomMark globally to use it from the terminal, or locally to use it inside a JavaScript or Node.js project.

## 1. Global (Command Line)
Install SomMark globally to use the `sommark` command:
```bash
npm install -g sommark
```

Check that it installed correctly:
```bash
sommark --version
```

## 2. Local (Project API)
Add SomMark to your project:
```bash
npm install sommark
```

### Quick Start
```javascript
import SomMark from "sommark";

const sm = new SomMark({
  src: "[h1]Hello World[end:h1]",
  format: "html"
});

const output = await sm.transpile();
console.log(output);
// <h1>Hello World</h1>
```
