# Installation

You can install SomMark globally to use it in your terminal, or locally to use it inside your javascript or nodejs project.

## 1. Global (Command Line)
Install SomMark globally to use the `sommark` command.
```bash
npm install -g sommark
```

To verify it is working, check the version:
```bash
sommark --version
```

## 2. Local (Project API)
Add SomMark to your project as a dependency:
```bash
npm install sommark
```

### Basic Engine Usage
Import the `SomMark` class to get started:
```javascript
import SomMark from "sommark";

const sm = new SomMark({
  src: "[h1]Hello World[end]",
  format: "html"
});

const output = await sm.transpile();
console.log(output);
```
