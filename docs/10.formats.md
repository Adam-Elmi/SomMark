# Supported Output Formats

SomMark can convert `.smark` files into multiple formats. Each format serves a different purpose:

---

## 1. HTML
The default format. Used for web pages and components.

```javascript
import { transpile } from "sommark";
const html = await transpile({ src, format: "html" });
```

---

## 2. Markdown
Converts SomMark to standard Github-flavored Markdown.

```javascript
const md = await transpile({ src, format: "markdown" });
```

---

## 3. MDX
Converts to MDX format, suitable for React-based workflows.

```javascript
const mdx = await transpile({ src, format: "mdx" });
```

---

## 4. JSON
Exports the document structure as a JSON object, useful for programmatic analysis.

```javascript
const json = await transpile({ src, format: "json" });
```

---

## 5. Plain Text
Strips all formatting and returns only the text content.

```javascript
const text = await transpile({ src, format: "text" });
```

---

> [!Note]
> Only **one output format** can be used per command.
> You can **print to console** instead of saving using the `-p` flag:
> `sommark --html -p file.smark`
