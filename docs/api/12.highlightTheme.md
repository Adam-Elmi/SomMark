# highlightTheme

The currently selected Highlight.js theme.

**Syntax:** `mapper.highlightTheme` (String)

Default: `"atom-one-dark.min"`.

```js
import { Mapper } from "sommark";

const mapper = new Mapper();
console.log(mapper.highlightTheme); // "atom-one-dark.min"
mapper.highlightTheme = "github";
```
