# clear

The `clear` method removes all registered block and inline mappings from the mapper instance. This effectively resets the mapper to an empty state, allowing for a fresh set of registrations.

**Syntax:** `mapper.clear()`

### Example: Resetting a Mapper
```js
import { HTML } from "sommark";

// Clear all standard HTML mappers
HTML.clear();

console.log(HTML.outputs.length); // 0
```

> [!CAUTION]
> This will remove **every** registered output, including standard ones if called on an inherited mapper. Use with care.
