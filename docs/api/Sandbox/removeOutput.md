# removeOutput()

Unregisters a dynamic block, removing it from the output mappings.

---

**Syntax:**
```js
SomMark.removeOutput(id);
```

**Usage:**
```javascript
static ${
  // Unregisters the dynamic [alert] block
  SomMark.removeOutput("alert");
}$
```
