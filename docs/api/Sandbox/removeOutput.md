# removeOutput()

Unregisters a dynamic tag, removing it from the output mappings.
---

**Syntax:**
```js
SomMark.removeOutput(id);
```

**Usage:**
```javascript
static ${
  // Unregisters the dynamic [alert] tag
  SomMark.removeOutput("alert");
}$
```
