# includesId()

Checks if any of the provided tag names/IDs are registered in the active environment.
---

**Syntax:**
```js
SomMark.includesId(ids);
```

**Usage:**
```javascript
static ${
  // Checks if the [alert] or [card] tags are active
  const hasTags = SomMark.includesId(["alert", "card"]);
}$
```
