# includesId()

Checks if any of the provided block names/IDs are registered in the active environment.
---

**Syntax:**
```js
SomMark.includesId(ids);
```

**Usage:**
```javascript
static ${
  // Checks if the [alert] or [card] blocks are active
  const hasBlocks = SomMark.includesId(["alert", "card"]);
}$
```
