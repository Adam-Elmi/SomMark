# removeOutput

Removes a registered mapping by its ID. Sometimes you may want to remove a mapping that was registered by another mapper and replace it with your own.

**Syntax:** `mapper.removeOutput(id)`

-   **id** (string): The identifier of the mapping to remove.

```js
import { Mapper, HTML } from "sommark";

const myMapper = new Mapper();
myMapper.outputs = HTML.outputs; 

// Remove the default 'Block' mapping
myMapper.removeOutput("Block");

// Register a new 'Block' mapping
myMapper.register("Block", ({ content }) => {
  return myMapper.tag("div").body(content);
});
```

Or 

```js
import { Mapper, HTML } from "sommark";

const myMapper = new Mapper();

// Remove the default 'Block' mapping
HTML.removeOutput("Block");

myMapper.outputs = HTML.outputs; 

// Register a new 'Block' mapping
myMapper.register("Block", ({ content }) => {
  return myMapper.tag("div").body(content);
});
```
