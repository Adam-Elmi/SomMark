# list

Generates an HTML list (`<ul>` or `<ol>`) from data.

**Syntax:** `mapper.list(data, as)`

-   **data** (string | Array): The list items.
-   **as** (string): "ul" (unordered) or "ol" (ordered). Default: "ul".

```javascript
const html = mapper.list("Item 1\nItem 2", "ul");
// <ul><li>Item 1</li><li>Item 2</li></ul>
```
