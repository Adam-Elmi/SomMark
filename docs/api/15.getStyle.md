# getStyle

The `getStyle` method retrieves the combined CSS content of the currently selected highlight theme and any additional custom styles added via `addStyle`.

**Type:** `Function`

**Returns:**
- `String` - The complete CSS string.

### Example

```javascript
import { HTML } from "sommark";

// Get the current CSS to inject manually
const styles = HTML.getStyle();
console.log(styles);
```
