# clone

The `clone` method creates a deep copy of the current mapper instance. This is useful for creating isolated mapper states when you need to process different documents with slightly different configurations without affecting the original instance.

**Syntax:** `const newMapper = mapper.clone()`

-   **Returns**: A new instance of the same Mapper class.

### What is Cloned?
-   **`outputs`**: All registered blocks and inlines (options are isolated).
-   **`styles`**: All accumulated CSS styles.
-   **`pageProps`**: A deep copy of the document metadata (title, charset, etc.).
-   **`extraProps`**: A copy of the globally recognized argument keys.
-   **`customHeaderContent`**: All raw header content added via `setHeader`.

### Example: Isolation
Create a modified clone of a base mapper for a specific task.

```js
import { HTML } from "sommark";

// Create an isolated copy
const specializedHTML = HTML.clone();

// Modify the clone without affecting the global HTML mapper
specializedHTML.addStyle(".my-card { border: 1px solid red; }");
specializedHTML.pageProps.pageTitle = "Specialized Page";
```
