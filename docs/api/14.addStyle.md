# addStyle

Registers custom CSS to be included in the final output. When inside a `register()` callback, you should access it via `this.addStyle`.

**Syntax:** `this.addStyle(css)`

-   **css** (string | object): Raw CSS string or a nested object of rules.

### Example: Dynamic Styling in a Block
```javascript
HTML.register("Box", function({ args, content }) {
  // Add a one-time global style for this component
  this.addStyle(".my-box { padding: 20px; border: 1px solid #ccc; }");
  
  return this.tag("div").attributes({ class: "my-box" }).body(content);
});
```

### Example: Object Syntax
```javascript
HTML.addStyle({
  ".btn": { 
    background: "blue",
    color: "white"
  }
});
// Becomes: .btn{background:blue;color:white;}
```
