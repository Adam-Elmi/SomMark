# header

A read-only property that returns the generated HTML meta tags and document metadata (like `<title>`, `<meta charset>`, etc.) based on `pageProps`.

**Syntax:** `mapper.header` (String)

> [!NOTE]
> For the HTML mapper, this is automatically used during `transpile()` when `includeDocument` is true, combined with any added styles.

```javascript
console.log(mapper.header);
// <meta charset="UTF-8">
// <meta name="viewport" ...>
// <title>SomMark Page</title>
```
