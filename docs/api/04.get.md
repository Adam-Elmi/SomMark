# get

Get the mapping object for a specific tag ID. This is useful for retrieving existing renderers to wrap them or move them to a new mapper.

**Syntax:** `mapper.get(id)`

-   **id** (string): The identifier to search for.
-   **Returns**: A mapping object or `null`.

### Return Structure
The returned object (if found) has the following properties:
- `id` (string | Array): The registered identifier(s).
- `render` (function): The bound renderer function.
- `options` (object): The registration options.

### Example: Wrapping an existing renderer
```javascript
const originalCode = HTML.get("Code");

if (originalCode) {
  HTML.register("MyCode", function(data) {
    // Custom logic before calling the original
    console.log("Rendering code block...");
    return originalCode.render.call(this, data);
  }, originalCode.options);
}
```

> [!TIP]
> Use `mapper.inherit()` if you want to copy **all** mappings from one mapper to another.
