# removeOutput

Removes a registered mapping by its ID. This is typically used to override or disable default blocks when inheriting from other mappers.

**Syntax:** `mapper.removeOutput(id)`

-   **id** (string): The identifier of the mapping to remove.

### Example: Overriding a Default Block
Inherit from the standard HTML mapper but replace the `bold` implementation.

```js
import { Mapper, HTML } from "sommark";

const myMapper = new Mapper();
myMapper.inherit(HTML); 

// Remove the default 'bold' mapping
myMapper.removeOutput("bold");

// Register your own alternative
myMapper.register("bold", function({ content }) {
  return `<b>${content}</b>`;
});
```

> [!IMPORTANT]
> To prevent duplicate IDs, `mapper.register()` automatically calls `removeOutput()` for the IDs being registered. You only need to call this manually if you want to cleanly delete a block without replacing it.
