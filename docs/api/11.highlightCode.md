# highlightCode

Highlights a string of code using Highlight.js.

**Syntax:** `mapper.highlightCode(code, language)`

-   **code** (string): The source code.
-   **language** (string): The language name (e.g., "javascript"). Default: "plaintext".
-   **Returns**: Highligthed HTML string.

```js
import { Mapper } from "sommark";

const mapper = new Mapper();
const html = mapper.highlightCode("const x = 1;", "javascript");
```
