## highlightCode(code, language)

Optional property used to enable syntax highlighting for code blocks.

**Initial value**

* `null`

It is designed to be assigned a function that integrates with a syntax highlighting library such as Highlight.js. If it remains `null`, code blocks are rendered without highlighting.

**Expected signature**

* `code` — Raw code string
* `language` — Language name
* Returns highlighted HTML string

---

### Example Integration (Highlight.js)

```js
import hljs from "highlight.js";

function highlightCode(code, language = "plaintext") {
  if (!hljs.getLanguage(language)) {
    language = "plaintext";
  }

  return hljs.highlight(code, { language }).value;
}

export { highlightCode };
```

Then assign it:

```js
this.highlightCode = highlightCode;
```
