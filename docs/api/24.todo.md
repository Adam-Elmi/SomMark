# todo

Parses a todo item state.

**Syntax:** `mapper.todo(checked)`

-   **checked** can be a string or boolean value: 
  - if checked equals to "x" or "done" or true then `todo` method will return `true`. Else it will return `false`
> ![NOTE]
> 'done' can be writen in uppercase or lowercase or mixed, it is not a case-sensitive.
-   **Returns**: `true` or `false`.

```javascript
const isDone = mapper.todo("x"); // true
// or 
const isDone2 = mapper.todo("done"); // true
const isPending = mapper.todo("-"); // false
```
