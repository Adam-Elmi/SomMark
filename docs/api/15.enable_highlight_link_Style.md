# enable_highlight_link_Style

Controls whether SomMark injects a `<link rel="stylesheet">` tag for the Highlight.js theme.

**Syntax:** `mapper.enable_highlight_link_Style` (boolean)

Default: `false`.

### What does it do?
When set to `true`, SomMark adds a `<link>` tag pointing to the selected Highlight.js theme CSS file in `node_modules`.

### When to use it?
Enable this if you prefer loading the theme as an external file rather than having the styles inlined in the `<head>`. This is useful if:
-   You are serving the `node_modules` directory statically.
-   You want to cache the CSS file separately.

### Example: Enable Link Injection
```js
import { Mapper } from "sommark";

const mapper = new Mapper();

// Enable link tag
mapper.enable_highlight_link_Style = true;

// Set the theme
mapper.highlightTheme = "github";

// Output HTML will include:
// <link rel="stylesheet" href="node_modules/highlight.js/styles/github.css">
```
