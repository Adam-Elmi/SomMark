# comment

The `comment` method defines how comments in the source SMark file should be rendered in the final output.

**Syntax:** `mapper.comment(text)`

-   **text** (string): The raw comment content.
-   **Returns**: A string representing the comment in the target format.

### Default Behavior
In the base `Mapper` class, this method returns an empty string (comments are stripped by default).

### Implementation Examples

#### HTML Mapper
```javascript
comment(text) {
  return `<!--${text.replace("#", "").trim()}-->\n`;
}
```

#### Markdown Mapper
```javascript
comment(text) {
  return `<!--${text.replace("#", "").trim()}-->\n`;
}
```

> [!NOTE]
> SomMark comments are defined with a hash `#` in the source. This hash is passed as part of the `text` parameter.
