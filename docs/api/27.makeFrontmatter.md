## makeFrontmatter(entries)

Creates a Markdown/MDX frontmatter block from an object of metadata.

**Parameters**

* `entries: object` — Key-value pairs for frontmatter fields.

**Returns**

* `string` — Frontmatter formatted as Markdown metadata block.

**Example**

```js
makeFrontmatter({
	title: "My Document",
	tags: ["mdx", "astro"],
	author: "Adam"
});
```

Produces:

```
---
title: "My Document"
tags:
  - "mdx"
  - "astro"
author: "Adam"
---
```
