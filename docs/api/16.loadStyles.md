# loadStyles

Asynchronously loads the Highlight.js theme CSS and combines it with any custom styles registered via `addStyle`.

**Syntax:** `mapper.loadStyles()`

-   **Returns**: `Promise<string>` (All combined CSS content).

### What does it do?
It reads the CSS file for the current `highlightTheme` (if code exists) and appends all strings from the `mapper.styles` array. It returns the raw CSS string.

### When to use it?
SomMark calls this internally when `enable_highlight_style_tag` is true. You can call it manually if you want to:
-   Get the CSS string for Server-Side Rendering (SSR).
-   Inject the styles into a different part of your document.
-   Write the styles to a `.css` file during a build process.

### Example: Manual Usage
```js
import { Mapper } from "sommark";

const mapper = new Mapper();
mapper.addStyle(".my-class { color: red; }");

// Get all CSS (Theme + Custom)
const css = await mapper.loadStyles();
console.log(css);
// Output:
// ...highlight.js css...
// .my-class { color: red; }
```
