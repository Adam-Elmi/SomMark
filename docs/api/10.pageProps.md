# pageProps

A configuration object that stores metadata for the HTML page header, such as the title, charset, and favicon. This object is used by the `header` property to generate meta tags.

**Syntax:** `mapper.pageProps` (Object)

### Default Structure
```json
{
  "pageTitle": "SomMark Page",
  "tabIcon": { "type": "image/x-icon", "src": "" },
  "charset": "UTF-8",
  "viewport": "width=device-width, initial-scale=1.0",
  "httpEquiv": { "X-UA-Compatible": "IE=edge" }
}
```

### Example: Customizing Metadata
```js
import { HTML } from "sommark";

// Change Title
HTML.pageProps.pageTitle = "My Documentation";

// Set Favicon
HTML.pageProps.tabIcon.src = "/favicon.ico";
HTML.pageProps.tabIcon.type = "image/png";

// Add custom meta tag
HTML.pageProps.httpEquiv["refresh"] = "30";
```

> [!NOTE]
> These properties only affect the output when `includeDocument` is set to `true` during the transpilation process.
