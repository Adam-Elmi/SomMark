# pageProps

A configuration object that stores metadata for the HTML page header, such as the title, charset, and favicon.

**Syntax:** `mapper.pageProps` (Object)

### Default Structure
```json
{
  "pageTitle": "SomMark Page",
  "tabIcon": {
    "type": "image/x-icon",
    "src": ""
  },
  "charset": "UTF-8",
  "viewport": "width=device-width, initial-scale=1.0",
  "httpEquiv": { "X-UA-Compatible": "IE=edge" }
}
```

### Usage
You can modify these properties to customize the `<head>` of your generated HTML.

```js
import { Mapper } from "sommark";

const mapper = new Mapper();

// Change Title
mapper.pageProps.pageTitle = "My Documentation";

// Set Favicon
mapper.pageProps.tabIcon.src = "/favicon.ico";
mapper.pageProps.tabIcon.type = "image/png";

// Add specific meta tag
mapper.pageProps.httpEquiv["refresh"] = "30";
```
> [!NOTE]
> Adjust `mapper.pageProps.tabIcon.src` path to where output file is generated, else favicon will not be displayed.
