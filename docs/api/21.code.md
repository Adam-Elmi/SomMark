# code

Formats a code block with syntax highlighting.

**Syntax:** `mapper.code(args, content)`

-   **args** (Array): Arguments from the block (first arg usually language).
-   **content** (string): The raw code content.

```javascript
mapper.register("Code", ({ args, content }) => {
  return mapper.code(args, content); // Returns <pre><code class="...">...</code></pre>
});
```
