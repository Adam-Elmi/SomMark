# parseList

Parses a raw string into a hierarchical list structure based on indentation.

**Syntax:** `mapper.parseList(data, indentSize)`

-   **data** (string | Array): The raw list content.
-   **indentSize** (number): Spaces per indent level. Default: 2.
-   **Returns**: Array of nodes `{ text, children }`.

```javascript
const tree = mapper.parseList("Item 1\n  Subitem 1");
console.log(tree);
```
