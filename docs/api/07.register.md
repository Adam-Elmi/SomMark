# register

Registers a new tag renderer. This is the main method to extend SomMark.

**Syntax:** `mapper.register(id, renderer, options)`

-   **id** (string | Array\<string>): The tag name(s) (e.g., "Card", ["b", "bold"]).
-   **renderer** (function): A function receiving `{ args, content }`. Must return a string.
-   **options** (object): Optional settings (e.g., `{ escape: false }`).

### Example: HTML (Standard)
Use `.body()` when your tag needs to hold text or other elements inside it.

```js
import { Mapper } from "sommark";

const mapper = new Mapper();
mapper.register("Alert", ({ content }) => {
  return mapper.tag("div")
    .attributes({ class: "alert" })
    .body(content);
});
```

### Example: HTML (Self-Closing)
Use `.selfClose()` for tags that are empty and don't wrap content, like images or inputs.

```js
import { Mapper } from "sommark";

const mapper = new Mapper();
mapper.register("Image", ({ args }) => {
  return mapper.tag("img")
    .attributes({ src: args[0] })
    .selfClose();
});
```

### Example: Markdown
Use `mapper.md` helpers to create valid Markdown elements easily.

```js
import { Mapper } from "sommark";

const mapper = new Mapper();
mapper.register("Heading", ({ args, content }) => {
  // Use mapper.md for markdown helpers
  return mapper.md.heading(content, (args && args[0]) || 1);
});
```
