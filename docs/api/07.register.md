# register

Registers a new tag renderer. This is the main method to extend SomMark.

**Syntax:** `mapper.register(id, renderer, options)`

-   **id** (string | Array\<string>): The tag name(s) (e.g., `"Card"`, `["b", "bold"]`).
-   **renderer** (function): A function receiving `{ args, content }`. 
    -   **Context**: The function is bound to the `mapper` instance, so you can use `this.tag()`, `this.safeArg()`, etc.
-   **options** (object): Optional settings:
    -   `escape` (boolean): Whether to escape the content before passing it to the renderer. Default: `true`.
    -   `type` (string): The block category for validation and parsing. Common values: `"Block"`, `"AtBlock"`, `"any"`.
    -   `rules` (object): Advanced validation rules (e.g., `{ is_self_closing: true }`).

### Example: Basic Expansion
Use `this.tag()` to build HTML elements cleanly.

```js
import { HTML } from "sommark";

HTML.register("Alert", function({ content }) {
  return this.tag("div")
    .attributes({ class: "alert" })
    .body(content);
});
```

### Example: Multiple IDs & safeArg
Registering multiple aliases and handling arguments safely.

```js
import { HTML } from "sommark";

HTML.register(["img", "image"], function({ args }) {
  const src = this.safeArg(args, 0, "src", null, null, "");
  return this.tag("img")
    .attributes({ src })
    .selfClose();
});
```

### Example: Complex Markdown with Type
Using the `AtBlock` type for specialized command-like blocks.

```js
import { Markdown } from "sommark";

Markdown.register("Code", function({ args, content }) {
  const lang = this.safeArg(args, 0, "lang", null, null, "text");
  return this.md.codeBlock(content, lang);
}, { type: "AtBlock", escape: false });
```

### Advanced Rules: Argument Validation
The `rules` option allows you to enforce strict requirements on how your block is used.

- `includes`: (Array) List of allowed argument names. Any other arguments will throw an error.
- `force`: (Array) List of required argument names. Missing these will throw an error.
- `is_self_closing`: (Boolean) If true, the block must be closed immediately (e.g., `[Image = src: ...][end]`).

```js
HTML.register("Card", function({ args, content }) {
  const title = this.safeArg(args, 0, "title", null, null, "Untitled");
  return this.tag("div")
    .attributes({ class: "card", title })
    .body(content);
}, {
  type: "Block",
  rules: {
    includes: ["title", "id", "class"], // Only these args allowed
    force: ["title"]                   // Title MUST be provided
  }
});
```

> [!TIP]
> Always use `function()` instead of arrow functions `() => {}` if you need to access `this` helpers like `this.tag` or `this.safeArg`.
