# includesId

Checks if any of the provided IDs are already registered in the mapper's `outputs`. This is useful for plugins or feature detection within custom mappers.

**Syntax:** `mapper.includesId(ids)`

-   **ids** (Array\<string>): A list of IDs to check.
-   **Returns**: `true` if **any** of the provided IDs exist in the registry, otherwise `false`.

### Example: Feature Detection
```javascript
if (HTML.includesId(["Card", "Panel"])) {
  console.log("Found supporting components!");
}
```

### Usage in Register
```javascript
HTML.register("Wrapper", function({ content }) {
  if (this.includesId(["SpecialHeader"])) {
    // Logic that depends on another block being registered
  }
  return `<div>${content}</div>`;
});
```
