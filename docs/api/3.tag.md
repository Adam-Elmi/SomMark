# tag

The `tag` method returns a `TagBuilder` instance, allowing you to construct HTML elements programmatically and safely.

**Syntax:** `mapper.tag(tagName)`

-   **tagName** (string): The name of the HTML tag (e.g., "div", "span").

```js
import { Mapper } from "sommark";

const mapper = new Mapper();
const div = mapper.tag("div").body("Hello World");
// <div>Hello World</div>
```

---

## TagBuilder Methods

### `attributes(obj, ...keys)`
Adds HTML attributes to the tag.

-   **obj** (object): Key-value pairs of attributes.
-   **keys** (string[]): List of boolean attributes (e.g., "disabled").

```js
mapper.tag("input")
  .attributes({ type: "text", name: "username" }, "required")
  .selfClose()
// <input type="text" name="username" required>
```

### `props(propsList)`
Adds strictly typed React-like props.

-   **propsList** (Array | Object): Object(s) with `{ type, ...props }`.
-   **type**: "string" (normal attribute) or "other" (interpolated `{value}`).

```js
mapper.tag("MyComponent")
  .props({ type: "string", title: "Hello" })
  .props({ type: "other", count: 10 })
  .selfClose()
// <MyComponent title="Hello" count={10}>
```

### `body(nodes)`
Appends content inside the tag.

-   **nodes** (string): The inner HTML or text.

```js
mapper.tag("p")
  .body("Hello World")
// <p>Hello World</p>
```

### `selfClose()`
Marks the tag as self-closing (void element).

```js
mapper.tag("img")
  .attributes({ src: "pic.jpg" })
  .selfClose()
// <img src="pic.jpg" />
```
