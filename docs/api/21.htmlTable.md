# htmlTable

Generates an HTML table from data.

**Syntax:** `mapper.htmlTable(data, headers)`

-   **data** (Array | string): Array of rows, or a newline-separated string.
-   **headers** (Array\<string>): List of header titles.

```javascript
const headers = ["ID", "Name"];
const data = ["1, Adam", "2, Elmi"];

return mapper.htmlTable(data, headers);
```
