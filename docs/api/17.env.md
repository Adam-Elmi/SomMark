# env

Sets the runtime environment for resource loading (e.g., getting Highlight.js themes).

**Syntax:** `mapper.env` (String)

-   **Values**: `"node"` or `"browser"`
-   **Default**: `"node"`

### What does it do?
It tells SomMark how to load external files like syntax highlighting themes.
-   **"node"**: Uses `fs.readFile` (loading from the file system).
-   **"browser"**: Uses specific browser fetching logic (if configured) or skips file system operations.

### Example: Setting Browser Environment
```js
import { Mapper } from "sommark";

const mapper = new Mapper();

// Set to browser mode
mapper.env = "browser";

```
