# formatOutput

The `formatOutput` method is a hook that allows mappers to perform final transformations on the transpiled output and optionally wrap it in a full document structure (e.g., adding `<html>`, `<body>`, etc.).

**Syntax:** `mapper.formatOutput(output, includeDocument)`

-   **output** (string): The accumulated transpiled content from all nodes.
-   **includeDocument** (boolean): Whether to wrap the output in a full document structure.
-   **Returns**: The final formatted string.

### How it Works
This method is called by the transpiler as the very last step.
-   It provides a place to inject accumulated styles, scripts, and meta tags.
-   It allows for environment-specific formatting (e.g., minification or indentation).

### Example: HTML Mapper
The HTML mapper uses this hook to wrap the output in a full HTML document and inject the `<style>` block.

```javascript
formatOutput(output, includeDocument) {
  if (includeDocument) {
    const finalHeader = this.header;
    const styleTag = `<style>\n${this.styles.join("\n")}\n</style>`;
    
    return `<!DOCTYPE html>
<html>
${finalHeader}
${styleTag}
<body>
${output}
</body>
</html>`;
  }
  return output;
}
```

> [!TIP]
> Override this method if you need to add custom wrappers or perform global search-and-replace on the final transpiled string.
