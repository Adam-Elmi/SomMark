# registerHighlightTheme

The `registerHighlightTheme` method allows you to register custom syntax highlighting themes by providing their CSS content.

**Type:** `Function`

**Parameters:**
- `themes` (Object): An object where keys are theme names and values are the CSS content strings.

### Example

```javascript
import { HTML } from "sommark";

const draculaTheme = `pre code.hljs{...}`; // CSS String

HTML.registerHighlightTheme({
    "dracula": draculaTheme
});
```
