# tag

The `tag` method returns a `TagBuilder` instance, allowing you to construct HTML elements programmatically and safely. When inside a `register()` callback, you should access it via `this.tag()`.

**Syntax:** `this.tag(tagName)`

-   **tagName** (string): The name of the HTML tag (e.g., `"div"`, `"span"`).

```js
HTML.register("Greeting", function({ content }) {
  return this.tag("div").body(`Hello ${content}`);
});
```

---

## TagBuilder Methods

### `attributes(obj, ...keys)`
Adds HTML attributes to the tag.

-   **obj** (object): Key-value pairs of attributes.
-   **keys** (string[]): List of boolean attributes (e.g., `"disabled"`).

```js
this.tag("input")
  .attributes({ type: "text", name: "username" }, "required")
  .selfClose();
// <input type="text" name="username" required>
```

### `props(propsList)`
Adds strictly typed React-like props. (Mainly for custom component mappers).

```js
this.tag("MyComponent")
  .props({ __type__: "string", title: "Hello" })
  .selfClose();
// <MyComponent title="Hello">
```

### `body(nodes)`
Appends content inside the tag.

```js
this.tag("p").body("Hello World");
// <p>Hello World</p>
```

### `selfClose()`
Marks the tag as self-closing (void element).

```js
this.tag("img")
  .attributes({ src: "pic.jpg" })
  .selfClose();
// <img src="pic.jpg" />
```

---

## Usage in Register
Always use `function()` (not arrow functions) to access `this.tag`.

```js
HTML.register("Box", function({ content }) {
  return this.tag("div").body(content);
});
```
