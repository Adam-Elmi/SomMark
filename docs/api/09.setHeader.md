# setHeader

Appends custom HTML to the document head.

**Syntax:** `mapper.setHeader(htmlArray)`

-   **htmlArray** (Array\<string>): List of HTML strings.

```javascript
mapper.setHeader([
  '<meta charset="UTF-8">',
  '<title>My Page</title>'
]);
```

### `getCustomHeaderContent()`
Returns the accumulated custom header string.

```javascript
const content = mapper.getCustomHeaderContent();
```
