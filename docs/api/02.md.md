# md

The `md` property exposes an instance of the `MarkdownBuilder` class. When inside a `register()` callback, you should access it via `this.md`.

**Syntax:** `this.md` (MarkdownBuilder)

---

## Methods

### `heading(text, level)`
Creates a Markdown heading.

```js
const h1 = this.md.heading("Introduction", 1); // "\n# Introduction\n"
const h3 = this.md.heading("Details", 3);      // "\n### Details\n"
```

### `bold(text, is_underscore)`
Makes text bold.

```js
const strong = this.md.bold("Important"); // "**Important**"
```

### `italic(text, is_underscore)`
Makes text italic.

```js
const em = this.md.italic("Note"); // "*Note*"
```

### `codeBlock(code, language)`
Creates a fenced code block.

```js
const block = this.md.codeBlock("console.log('Hi')", "js");

// ```js
// console.log('Hi')
// ```
```

### `url(type, text, url, title)`
Creates a link or image.

-   **type** (string): `"image"` for `![]()`, empty for `[]()`.

```js
const link = this.md.url("", "Home", "/"); // "[Home](/)"
const img = this.md.url("image", "Logo", "logo.png"); // "![Logo](logo.png)"
```

### `table(headers, rows)`
Creates a Markdown table.

```js
const table = this.md.table(
  ["ID", "Name"],
  ["1, Alice", "2, Bob"]
);
```

### `todo(checked, text)`
Creates a task list item.

```js
const task = this.md.todo(true, "Buy milk"); // "- [x] Buy milk\n"
```

### `horizontal(format)`
Creates a horizontal rule.

```js
const hr = this.md.horizontal(); // "\n***\n"
```

### `escape(text)`
Escapes Markdown special characters.

```js
const safe = this.md.escape("[No Link]"); // "\[No Link\]"
```

---

## Usage in Register
Always use `function()` (not arrow functions) to access `this.md`.

```js
md.register("MyBlock", function({ content }) {
  return this.md.bold(content);
});
```
