# includesId

Checks if any of the provided IDs are already registered in the mapper.

**Syntax:** `mapper.includesId(ids)`

-   **ids** (Array\<string>): A list of IDs to check.
-   **Returns**: `true` if any ID exists, otherwise `false`.

## Multiple IDs
```javascript
if (mapper.includesId(["Card", "Panel"])) {
  console.log("One of these tags is already taken.");
}
```

## Single ID
```javascript
if (mapper.includesId("Card")) {
  console.log("This tag is already taken.");
}
```
