# outputs

A list storing all registered block and inline mappings.

**Syntax:** `mapper.outputs` (Array)

```js
console.log(mapper.outputs.length); // 5
// Each item is { id, render, options }
```

### Get outputs from default mappers
Get outputs from default mappers to avoid redefining common mappings.
```js
import { Mapper, HTML } from 'sommark';

const myMapper = new Mapper();
myMapper.outputs = HTML.outputs;
```

### Get outputs from other mappers
Get outputs from other mappers to avoid redefining common mappings.

```js
import { Mapper } from 'sommark';
import otherMapper from './otherMapper';

const myMapper = new Mapper();
myMapper.outputs = otherMapper.outputs;
// You can still register new outputs. It will not override existing mappings.
myMapper.register("Text", ({content}) => {
    return content;
});
```

### Get specific outputs
Get specific outputs from other mappers to avoid redefining common mappings.
```js
import { Mapper, HTML } from 'sommark';

const myMapper = new Mapper();
const codeOutput = HTML.get("code");
myMapper.register(codeOutput.id, codeOutput.render, codeOutput.options);
```
