## loadCss(env, filePath)

Loads CSS file content either from the **server (Node)** or **browser environment**.

**Parameters**

* `env: string` — Execution environment (`node` or `browser`).
  If not provided, it is automatically detected.
* `filePath: string` — Path or URL of the CSS file.

**Returns**

* `string` — CSS file content, or empty string if loading fails.

---

### Example Usage

```js
const css = await loadCss("node", "./styles/main.css");
```

or

```js
const css = await loadCss(undefined, "/styles/main.css");
```

---

### Behavior

**Node environment**

* Reads CSS file from filesystem.
* Validates:

  * File exists
  * File extension is `.css`

**Browser environment**

* Fetches CSS file using HTTP request.
* Validates response and file extension.

---

### Notes

* Automatically detects environment if `env` is not provided.
* Logs errors when:

  * File is missing
  * File extension is not `.css`
  * Fetch or read operation fails.
