# enable_highlight_style_tag

Controls whether SomMark automatically injects Highlight.js CSS styles into the HTML `<head>`.

**Syntax:** `mapper.enable_highlight_style_tag` (boolean)

Default: `true`.

### What does it do?
When set to `true` (default), SomMark automatically loads the CSS for the current syntax highlighting theme and injects it into a `<style>` block in the HTML `<head>`.

### Side Effects of Disabling
If set to `false`, the syntax highlighting CSS will not be injected.

### Example: Default (Enabled)
```js
import { Mapper } from "sommark";

const mapper = new Mapper();
// mapper.enable_highlight_style_tag is true by default
// Output HTML will include:
// <head>
//   <style>...css code...</style>
// </head>
```

### Example: Disabled
```js
import { Mapper } from "sommark";

const mapper = new Mapper();
// Disable automatic style injection
mapper.enable_highlight_style_tag = false;

```
