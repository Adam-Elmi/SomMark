# selectHighlightTheme

The `selectHighlightTheme` method sets the active syntax highlighting theme. The theme must be previously registered (or be the default `atom-one-dark`).

**Type:** `Function`

**Parameters:**
- `themeName` (String): The name of the theme to select.

### Example

```javascript
import { HTML } from "sommark";

// Select the default theme
HTML.selectHighlightTheme("atom-one-dark");

// Select a custom theme (must be registered first)
HTML.selectHighlightTheme("dracula");
```
