# escapeHTML

Escapes special characters (`<`, `>`, `&`, `"`, `'`) to prevent XSS attacks and maintain HTML integrity. When inside a `register()` callback, you should access it via `this.escapeHTML`.

**Syntax:** `this.escapeHTML(string)`

-   **string**: The raw string to escape.
-   **Returns**: The escaped safe string.

### Example: Manual Escaping
```javascript
HTML.register("SafeText", function({ content }) {
  const safeContent = this.escapeHTML(content);
  return `<span>${safeContent}</span>`;
});
```

> [!TIP]
> SomMark automatically escapes content unless you set `{ escape: false }` in the `register()` options.
