# escapeHTML

Escapes special characters (`<`, `>`, `&`, `"`) to prevent XSS attacks and broken HTML.

**Syntax:** `mapper.escapeHTML(string)`

-   **string**: The raw string to escape.
-   **Returns**: The safe HTML string.

```javascript
const safe = mapper.escapeHTML("<script>alert('xss')</script>");
console.log(safe);
// &lt;script&gt;alert('xss')&lt;/script&gt;
```
