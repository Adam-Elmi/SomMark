## raw_js_imports(imports)

Creates multiple JavaScript import statements from an array of import objects.

**Parameters**

* `imports: array` — Each item should contain:

  * `name` — Variable name to import as
  * `path` — File path of the module

**Returns**

* `string` — Combined import statements formatted for MDX or code generation.

**Example**

```js
raw_js_imports([
	{ name: "Intro", path: "../../Components/Intro.astro" },
	{ name: "Section", path: "../../Components/Section.astro" }
]);
```

Produces:

```js
import Intro from "../../Components/Intro.astro";
import Section from "../../Components/Section.astro";

```

**Notes**

* Expects an array of import objects.
* Automatically adds `\n` between imports.
* Adds `\n\n` after the last import to satisfy MDX parsing requirements.
* Returns empty string if inputs are missing.
