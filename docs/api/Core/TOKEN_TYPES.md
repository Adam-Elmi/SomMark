# TOKEN_TYPES

Defines the complete list of lexical tokens generated by the SomMark lexer.
---

**Syntax:**
```js
import { TOKEN_TYPES } from "sommark";
```

**Usage:**
```js
import { TOKEN_TYPES } from "sommark";

if (token.type === TOKEN_TYPES.EXCLAMATION_MARK) {
  console.log("Found self-closing shorthand block!");
}
```

---

### Token Reference Table

| Token Type | Representation | Description |
|:---|:---|:---|
| `OPEN_BRACKET` | `[` | Starts a block header |
| `CLOSE_BRACKET` | `]` | Ends a block header |
| `END_KEYWORD` | `end` | The reserved block-closing keyword |
| `EXCLAMATION_MARK` | `!` | Self-closing block shorthand marker |
| `IDENTIFIER` | `h1`, `div` | Target name of blocks, slots, or modules |
| `EQUAL` | `=` | Assigns block argument values |
| `KEY` | `class`, `id` | Name of a block or inline argument |
| `VALUE` | `blue`, `bold` | Literal value assigned to an argument key |
| `QUOTE` | `"`, `'` | Wraps argument strings |
| `COLON` | `:` | Separates keys and values |
| `COMMA` | `,` | Separates multiple block arguments |
| `THIN_ARROW` | `->` | Maps inline content to an inline tag |
| `OPEN_PAREN` | `(` | Starts inline content or tag definitions |
| `CLOSE_PAREN` | `)` | Ends inline content or tag definitions |
| `OPEN_AT` | `@_` | Starts an At-Block boundary |
| `CLOSE_AT` | `_@` | Ends an At-Block boundary |
| `SEMICOLON` | `;` | Separates At-Block headers from their body |
| `PREFIX_P` | `p{user}` | Dynamic public placeholder layer |
| `PREFIX_V` | `v{item}` | Local variables layer |
| `PREFIX_JS` | `js{obj}` | Native JavaScript object/array layer |
| `STATIC_KEYWORD` | `static` | Evaluates isolated JS once during compile-time |
| `RUNTIME_KEYWORD` | `runtime` | Marks code blocks for execution at target runtime |
| `LOGIC` | `${ code }$` | Raw JavaScript logic sandboxed inside the VM |
| `FOR_EACH` | `for-each` | Standard block iteration loop |
| `SLOT_KEYWORD` | `slot` | Reserves template placeholder content injection |
| `IMPORT` | `import` | The import statement for files and modules |
| `USE_MODULE` | `$use-module` | Resolves imported template elements |
| `COMMENT` | `# text` | Single-line comment |
| `COMMENT_BLOCK` | `### multiline ###` | Multi-line comment block |
| `ESCAPE` | `\char` | Literalizes structural characters |
| `TEXT` | `Hello World` | Standard plain text content |
| `WHITESPACE` | ` `, `\n` | Spaces, tabs, and newlines |
| `EOF` | `""` | The absolute end of the token stream |

---

### Example: Structural Block Tokenization

Tokenizing a block with custom styling arguments:

```javascript
import { lexSync } from "sommark";

const tokens = lexSync('[div = class: "card"]Content[end]');
console.log(tokens.map(t => t.type));
/*
Output:
[
  "OPEN_BRACKET",   // [
  "IDENTIFIER",     // div
  "WHITESPACE",     //  
  "EQUAL",          // =
  "WHITESPACE",     //  
  "KEY",            // class
  "COLON",          // :
  "WHITESPACE",     //  
  "QUOTE",          // "
  "VALUE",          // card
  "QUOTE",          // "
  "CLOSE_BRACKET",  // ]
  "TEXT",           // Content
  "OPEN_BRACKET",   // [
  "END_KEYWORD",    // end
  "CLOSE_BRACKET",  // ]
  "EOF"             // End of File
]
*/
```

### Example: Logic Block Tokenization

Tokenizing compile-time static script blocks:

```javascript
import { lexSync } from "sommark";

const tokens = lexSync('static ${ "Hello Smark" }$');
console.log(tokens.map(t => ({ type: t.type, value: t.value })));
/*
Output:
[
  { type: "STATIC_KEYWORD", value: "static" },
  { type: "WHITESPACE",     value: " " },
  { type: "LOGIC",          value: " \"Hello Smark\" " },
  { type: "EOF",            value: "" }
]
*/
```

### Example: Self-Closing Block Tokenization

Tokenizing a self-closing primitive block:

```javascript
import { lexSync } from "sommark";

const tokens = lexSync('[string = key: "age", value: "25" !]');
console.log(tokens.map(t => t.type));
/*
Output:
[
  "OPEN_BRACKET",      // [
  "IDENTIFIER",        // string
  "WHITESPACE",        //  
  "EQUAL",             // =
  "WHITESPACE",        //  
  "KEY",               // key
  "COLON",             // :
  "WHITESPACE",        //  
  "QUOTE",             // "
  "VALUE",             // age
  "QUOTE",             // "
  "COMMA",             // ,
  "WHITESPACE",        //  
  "KEY",               // value
  "COLON",             // :
  "WHITESPACE",        //  
  "QUOTE",             // "
  "VALUE",             // 25
  "QUOTE",             // "
  "WHITESPACE",        //  
  "EXCLAMATION_MARK",  // !
  "CLOSE_BRACKET",     // ]
  "EOF"                // End of File
]
*/
```
