# getHighlightTheme

Generates the HTML `<link>` tag for importing a Highlight.js theme.

**Syntax:** `mapper.getHighlightTheme(themeName)`

-   **themeName** (string): The theme to load.
-   **Returns**: The validated theme name (string).

```js
import { Mapper } from "sommark";

const mapper = new Mapper();
const theme = mapper.getHighlightTheme("github");
console.log(theme); // "github"

const badTheme = mapper.getHighlightTheme("invalid-theme");
console.log(badTheme); // "atom-one-dark.min"
```
