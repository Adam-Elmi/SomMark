# styles

An array containing all accumulated custom CSS strings registered via `addStyle()`.

**Syntax:** `mapper.styles` (Array\<string>)

```javascript
HTML.addStyle(".text { color: red; }");

console.log(HTML.styles);
// [".text { color: red; }"]
```

> [!NOTE]
> For the HTML mapper, these styles are automatically joined and injected into a `<style>` block when `includeDocument` is true.
