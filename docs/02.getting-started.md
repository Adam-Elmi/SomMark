# Getting Started with SomMark

Follow these steps to create your first SomMark file and convert it to HTML.

---

## 1. Create a SomMark File

Create a new file named `hello.smark` with the following content:

```ini
[Block]
Hello, SomMark! [a = href: https\://example.com]Visit Website[end]
[end]
```

---

## 2. Compile via CLI

If SomMark is installed globally, run:

```bash
# Print HTML to console
sommark --html hello.smark
```

To save the output to a file:

```bash
# Save HTML as 'output.html'
sommark --html hello.smark -o output.html
```

> [!TIP]
> Use the `--print` or `-p` flag to print output instead of writing to a file.

```bash
# Print HTML to console
sommark --html -p hello.smark
```
or 

```bash
# Print HTML to console
sommark --html --print hello.smark
```
---

## 3. Compile via Code (JavaScript / Node.js)

If using SomMark programmatically, the `transpile` function is the recommended entry point:

```javascript
import { transpile } from "sommark";

const source = `
[Block]
Hello World!
[end]
`;

const html = await transpile({
  src: source,
  format: "html"
});

console.log(html);
```

* `src` → your SomMark content
* `format` → output type (`html`, `markdown`, `mdx`, `json`, `text`)
* `transpile()` → converts SomMark to the desired format

> [!Tip]
> This approach is useful if you want to integrate SomMark into your app or use custom mappers.
