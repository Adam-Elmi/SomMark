# Getting Started

This guide will help you write your first SomMark file and convert it to HTML.

## 1. Create a File
Create a new file named `hello.smark`.

```ini
[Block]
Hello, SomMark!
(Visit Website)->(Link: https://example.com)
[end]
```

## 2. Compile via CLI
If you installed SomMark globally, run this command in your terminal:

```bash
sommark --html hello.smark
```
This will print the HTML output to your console. To save it to a file:
```bash
sommark --html hello.smark -o output
```

## 3. Compile via Code
If you are using javascript or Node.js, you can do this programmatically.

```javascript
import SomMark from "sommark";

const source = `
[Block]
Hello World!
[end]
`;

const smark = new SomMark({
  src: source,
  format: "html"
});

const html = await smark.transpile();
console.log(html);
```
