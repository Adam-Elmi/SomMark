# Installation

You can install SomMark globally to use the CLI, or locally to use it in your project.

## 1. Global Installation (CLI)
Install it globally if you want to run `sommark` commands from your terminal.

```bash
npm install -g sommark
```
**Verify:**
```bash
sommark --version
```

## 2. Local Installation (For Projects)
Install it locally if you want to use the API (e.g., `new Mapper()`) in your code.

```bash
npm install sommark
```

**Importing:**
```javascript
import { Mapper } from "sommark";
```
